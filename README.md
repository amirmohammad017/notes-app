# 📝 Notes App

A minimal notes app built with React and Vite — create, edit, and delete notes with smooth animations.

🔗 **Live Demo:** [notes-app-nine-blush.vercel.app](https://notes-app-nine-blush.vercel.app/)

---

## 🛠️ Tech Stack

- ⚛️ **React** + **Vite**
- 🎨 **Tailwind CSS v4**
- 🔀 **React Router v6**
- 🎞️ **Framer Motion** — enter/exit animations
- 💾 **localStorage** — no backend needed

---

## ✨ Features

- 📝 Create, edit, and delete notes
- 🎬 Smooth card animations on add/remove
- 💾 Data persists in browser (localStorage)
- 📱 Fully responsive

---

## 🚀 Getting Started
```bash
git clone https://github.com/amirmohammad017/notes-app.git
cd notes-app
npm install
npm run dev
